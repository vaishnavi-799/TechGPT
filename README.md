# AI/ML FAQ Chatbot

This is an NLP-based FAQ chatbot built using Flask, NLTK, and Scikit-learn.

## Features
- FAQ chatbot
- TF-IDF vectorization
- Cosine similarity
- NLP preprocessing
- Flask backend
- HTML/CSS frontend

## Run Project

Install libraries:

pip install -r requirements.txt

Run:

python app.py

Open browser:

http://127.0.0.1:5000
